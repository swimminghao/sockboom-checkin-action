## 使用方法

在`secrets`新建下列`Actions secrets`

- TOKEN
  - 获取在GitHub `Setting` -> `Developer settings` -> `Personal access tokens` 中获取
- EMAIL
  - SockBoom邮箱
- PASSWORD
  - SockBoom密码

> 最后必须要自己 `star` 一下才会开始运行，测试的的时候只要点两次就好了（取消star，然后再star）
